# book(gauss)-vue

图书管理系统前端项目

## How to run

https://github.com/HavEWinTao/Book-Gauss.git

### 1.安装Nodejs(v14.15.4)

下载对于版本的[Nodejs](https://nodejs.org/en/download/)

### 2.安装vue-cli(v4.5.11)

```
# 安装完Nodejs之后，命令行执行
npm install -g @vue/cli@4.5.11
```

### 3.下载依赖(第一次)
```
# 进入项目文件夹，执行
npm install
```

### 4.运行项目

```
npm run serve
```
